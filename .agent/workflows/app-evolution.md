1. **Analyze**: Review functional specs and glossary.
2. **Design**: Create a design document in `d:\Repos\Derot-my-brain\Docs\Planning\` explaining the proposed changes.
3. **Approve**: Discuss with the USER.
4. **Update Status**: Add the new tasks to `d:\Repos\Derot-my-brain\Docs\Planning\Project-Status.md`.
5. **Implement**: Execute using `/backend-feature` and `/frontend-feature` workflows.
6. **Maintain**: Update `d:\Repos\Derot-my-brain\Docs\Reference\Glossary.md` if new terms were introduced.
